Guide d'utilisation du site

Partie Colors :

Pour commencer a dessiner : Choisissez une bordure(border) et un remplissage(fill) en cliquant sur le texte.

La couleur sélectionnée sera affichée dans le cercle en dessous.

Partie Shapes :

  • Ligne 1 : Rectangle - Carré - Triangle - Cercle

    • Pour dessiner, cliquez sur le bouton puis dessinez sur la zone de dessin.

  • Ligne 2 : Texte - Trait - Dessin - Petite Gomme

    • pour utiliser l'outils texte, sélectionnez une couleur dans "fill".

    pour utiliser l'outils dessin, sélectionnez une couleur dans "border".

  • Ligne 3 : Changement Bordure - Changement remplissage - Agrandir la taille - Diminuer la taille

    • pour changer la bordure/le remplissage, changer d'abord la couleur sélectionner dans les boutons border/fill puis cliquez sur le carré/rectangle

    Sélectionnez l'outils + ou - puis cliquez sur le carré/rectangle pour le modifier

  • Ligne 4 : Boutons de déplacement

    • Cliquez sur le bouton correspondant à la direction voulue puis cliquez sur la forme pour la déplacer

Partie Fonts:

Cliquez sur fonts puis choississez une des 3 polices pour l'outils Texte

Autres options

  • Vous pouvez redimensionner la zone de dessin
  • Si vous utilisez un outils de modification à l'intersection de 2 formes, vous pouvez modifier les 2 en même temps
  • Clear : Vide la zone de dessin
  • PNG : télécharge le dessin en format image
  • PDF : télécharge le dessin en format PDF
  • Save et recup : permet de sauvegarder puis récupérer un dessin plus tard
